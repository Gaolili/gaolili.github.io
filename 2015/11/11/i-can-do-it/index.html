使用hexo写博客

####导语

做了差不多快一年的iOS开发了，平常开发都是遇到问题就直接找度娘，不管如何实现，先拿过来看看可以把问题解决不，如果解决了，就放下不会再去研究人家是如何实现的。最近项目也不是那么忙了，有时间自己去看点东西，慢慢的有些领悟，原来当初是这样实现的，开始是使用印象笔记把看的东西记下来，后来看到一个好的博客地方，决定来折腾一番，好了，现在来总结一下hexo搭建

附：参考资料

  1. hexo官方文档

  2. Hexo你的博客

  3. 通过Hexo 在Github上搭建博客教程

安装Hexo 的前提，必须安装了 Node.js 和 git

##第一步：下载安装 Node.js

  • 按照提示一步步安装即可

##第二步：下载安装 git

##第三步： 安装 Hexo（前二步都完成之后）
在终端输入此命令

$ sudo  npm install -g hexo-cli

初始化建立站点

* $ hexo init  “your folder name”
* cd  “your folder name”
* npm install

##第四步：使用 Hexo (以下命令都是在你的 init 目录下执行的)

1. hexo new "title" 创建文章
2. hexo generate    生成静态页面
3. hexo server    本地启动
在浏览器输入<http://localhost:4000> 就可以看的效果

##第五步： hexo部署到github 上

  1. 登录github后，将鼠标点击github右上角“+”号，在下拉菜单上，选择“New repository”项，将跳到如下页面，填写库名称（username.github.io），勾选“Initialize this repository with a README”，点击“create repository”，即可完成创建库。
  2. 编辑 _config.yml 
deploy:
 type: github
 repository:  https://github.com/username/username.github.io
 branch: master

3.执行下列指令即可完成部署

hexo generate
hexo deploy

* 注意： 

每次修改本地文件后，需要 hexo generate 才能保存。每次使用命令时，都要在 H:\hexo 目录下
